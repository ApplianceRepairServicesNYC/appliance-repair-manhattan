## How to Download Images

### Option 1: Manual Download
1. Right-click each URL
2. Choose "Save Link As..." or "Download Linked File"
3. Save to the appropriate folder in your project

### Option 2: Using wget (Linux/Mac)
```bash
# Download all images at once
cd your-project-folder

# Logo
wget -O images/logo.svg "https://appliance-repair-group.com/wp-content/uploads/2024/03/appliance-repair-group-logo.svg"

# Hero background
wget -O images/backgrounds/hero-bg.jpg "https://appliance-repair-group.com/wp-content/uploads/2024/03/appliance-repair-group-appliance-repair-man-in-overalls-repairs-a-refrigerator-focusing-on-its-inner-components-with-a-set-of-tools.jpg"

# etc...
```

### Option 3: Bulk Download Script
Save this as `download-images.sh` and run it:

```bash
#!/bin/bash
cd "$(dirname "$0")"

# Create directories
mkdir -p images/{icons,services,brands,backgrounds,reviews}

# Download images using curl
curl -o images/logo.svg "https://appliance-repair-group.com/wp-content/uploads/2024/03/appliance-repair-group-logo.svg"
curl -o images/icons/favicon.png "https://appliance-repair-group.com/wp-content/uploads/2024/03/appliance-repair-group-favicon.png"
# Add more curl commands for each image...
```

## After Downloading

1. Replace the `.svg` placeholder files with the downloaded images
2. Update file extensions in `index.html` if needed (e.g., `.svg` to `.jpg`)
3. Optimize images for web using tools like:
   - TinyPNG (https://tinypng.com)
   - Squoosh (https://squoosh.app)
   - ImageOptim (Mac)

## Notes

- Some images may be copyrighted - ensure you have rights to use them
- Consider replacing with your own images for a unique look
- Optimize all images before deploying for faster load times
